一、Random forest prediction of Alzheimer’s disease using pairwise selection from time series data

1 研究目的

AD是渐变不可逆的，所以尽早地在AD发展的早期阶段进行潜在的药物治疗是很有必要的，所以测试治疗手段是否有助于诊断的预测是很有必要的。

而且另一方面PET扫描阳性是比较昂贵的，可以提供建议避免筛查失败。

2 AD预测的先例

  • 32 sMRI和amyloid PET的综合研究，MCI转变AD；
  • amyloid PET是更好的预测AD转变的指标比MRI（effect size :1.32 vs 0.77)，但是MRI中内嗅皮层的萎缩（effect size: 1.26)也是很好的biomarker；
  • amyloidosis 和 neurodegeneration biomarkers;
  • 个体最好的biomarker是FDG-PET去度量颞顶代谢的减退。

  • levels of markers from cerebral spinal fluid(CSF)

4 方法对比

官方提供了关于 lnear mixed effects model **和 **benchmark SVM 的方法可以做对比实验。

  • linear mixed effects model

    intercept (random effect) + slope(APOE)

  • SVM

    • feature: ADAS13, Ventricles
    • follow-up labeled: linear support vector machine
    • ADAS13/ventricle: linear support vector regression

  • random forest

    很好地应用了可量化和可分类的特征，易于训练和提供了对于每个变量的importance

    决策树的集成，每一个树划分输入空间最小化loss function,遍历迭代所有的树后，从而选择变量和划分节点得到最好的划分（即两个子集形成最小的整体差异）。

    • 为减少方差，随机选择n个特征的sqrt(n)的值为对应的分割点；最后回归的总体估计是决策子的平均，把他分类到占比重大的那个类。

    • 决策树的参数是树的个数以及一个叶子节点上最小的数据个数；这些值可以通过最小化OOB预测error得到，OOB预测的实现是通过过程的调整而非结果进行实现的。

【备注】：OOB预测误差（Out of bag error）计算：

Bootstrap每次约有1/3的样本不会出现在Bootstrap所采集的样本集合中,当然也就没有参加决策树的建立,那是不是意味着就没有用了呢,答案是否定的。我们把这1/3的数据称为袋外数据oob（out of bag）,它可以用于取代测试集误差估计方法.

袋外数据(oob)误差的计算方法如下：

​ 对于已经生成的随机森林,用袋外数据测试其性能,假设袋外数据总数为O,用这O个袋外数据作为输入,带进之前已经生成的随机森林分类器,分类器会给出O个数据相应的分类,因为这O条数据的类型是已知的,则用正确的分类与随机森林分类器的结果进行比较,统计随机森林分类器分类错误的数目,设为X,则袋外数据误差大小=X/O;这已经经过证明是无偏估计的,所以在随机森林算法中不需要再进行交叉验证或者单独的测试集来获取测试集误差的无偏估计。

5.3、相关结论

  • 相对影响因子比较大的几个变量是DX、RID、TIME_DELAY
  • 预测个体最好的因子是最近的诊断结果，时间是最大影响疾病发展的变量
  • AGE、RID、MIDTEMP、delta HIPPOCAMPUS是都很重要的因子
  • NC的预测准确99%，MCI的预测准确59%，AD的预测准确为29%，总体准确72%

预测结果比较低的原因是：因为要预测的时间跨度比较大，要1月到7年不等，这样时间节点的跨度大的话就会产生很高的误差。

（但是对应AD多分类算法中，大概是53%-96%的区间）

好处在于不用进行缺值插值操作，直接将时间因子作为变量

二、Disease progression timeline estimation for Alzheimer’s disease using discriminative event based modeling

1、研究目的

有助于理解疾病过程的动态变化、方便早期的诊断和病人分期的诊断。

对于AD疾病发展的变化预测，一方面可以去有助于进行预防性和支持性的治疗，方便改善生活质量和降低与护理和生活方式改变的成本；另一方面也可以去度量相应治疗的有效性。

2、AD预测的先例

  • 纵向数据研究有的是基于biomarker变化、有的是基于自身模型变化、有的是基于用户定制的混合模型变化研究，但是他们都比较依赖于相对比较大疾病数量的数据集，所以实现效果不是很好；
  • 为了规避受试者数量不够的问题，然后有人通过横截面数据的方式进行预测疾病的进程；有的采用症状的分型去预测特定的亚型，但这种方式粒度是比较粗和定性的；
  • EBM模型是基于biomarker变成异常的事件进行推断的，一个重要的假设是事件的顺序在一个数据集中变化是一致的，但是有点严格对于AD的研究，而且AD的biomarker的数量也比较大。

3、DEBM相应方法

3.1 EBM（Event-based model)

biomarker的值随疾病的严重程度单调增加/减少（有一定的噪音）。疾病的发展是由一系列的事件组成的，事件对应着一个新的biomarker变为异常的过程。Frontejin的EBM模型发现从受试者提取的数据集大部分遵循事件发生的顺序。

【基本假设】

控制人群中的所有biomarker是均匀的，且biomarker的值服从高斯分布，是可计算的；

而patient人群中的异常biomarker的值是服从均匀分布的。

所以将会得到一个高斯分布和一个均匀分布的混合模型，但是只有均匀分布中的参数可以被优化调整。

【基本算法】

  • S i是biomarkers对应的索引，表明他们变异常的相关顺序；

  • 是疾病变化的过程，N为数据集中每个受试者的biomarker数目；为对应关联S(i)生物标志物的第i个事件变为异常；

  • M个受试者中的横截面数据集（X），表明受试者的生物标志物的度量，由N维的生物标记物组成，对应一个EBM的概率公式可以通过得出，通过对应贝叶斯公式得出，但这里有个假设是**p(S)**是均匀分布的，

  • p(k|S）是一个受试者在事件发展过程中第k位置的先验概率，假设是相等的在每一个位置；最大化的第k个位置，表明受试者j的疾病阶段，这种方法通过识别受试者的严重程度去产生了离散的阶段，阶段的数量不一定与biomarkers的数量一致。

  • ,是观察在受试者j发生i事件的可能性，二是事件i不可能发生的可能性。

3.2 DBEM

3.2.1 biomarker progression

【不平凡性】

对于每一个biomarker求出对应事件前后发生的概率密度，其结果是不平凡的，原因：

  • biomarker的异常程度不能用临床状态（CN/AD）度量；
  • 临床状态的诊断不是二义的，可能会有其他的类别（MCI）

【贝叶斯分类器】

  • 【initial estimate】贝叶斯分类器通过每一个biomarker使用CN和AD状态进行训练，基于CN受试者没有biomarker异常和AD受试者全部受试者异常的假设 ，然后可以将受试者分谓很容易分类的CN、AD、和其他受试者，删去其他受试者，这样就能得到一个截断的高斯分布；

  • 【refine estimate】高斯混合分布，包括所有可用的数据

    在这里混合参数和高斯参数混合优化，直到混合参数收敛得到想要的结果。
