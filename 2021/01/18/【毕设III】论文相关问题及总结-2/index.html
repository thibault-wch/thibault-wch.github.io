Machine Learning with Biomedical Data

一、数据集限制

  • 并非所有visit都有数据；
  • 对于每一个受试者而言，可能有多个visit，但是visit的时间间隔是不规律的；
  • 每个受试者的visit数目也是不一样的；

二、Model 尝试

三、数据预处理

###　3.1、输入数据过滤

1、对于特征的筛选，如果这一列特征的空白值超过70%，则删去；

2、如果说某一列的值全是一样的value，那么也要删去；

通过以上步骤过滤掉了1500个特征，而只留下了大约337个特征（除了PTID_KEY和EXAMDATE）；

如果尝试用PCA，得到200个特征，但300多个是可以接受的，所以还是选择了这些；

3.2、输出数据过滤

按照PTID_KEY排序后进行筛选：

1、删去用户所有visit中所有列都为NaN的用户；

2、删去一个病人的所有visit中至少有一列为NaN的样本

四、对比实验

4.1、Baseline Algorithm(LOCF)

LOCF——Last Output Carry Forward

预测的对应值直接来自于最近的visit，基于自最后一次visit后用户的状态不会发生改变。

Predicting Alzheimer’s disease progression using deep recurrent neural networks

一、相关研究

1.1 过去AD预测探讨

  • 多因子回归模型

    linear/sigmoidal curve 受限于即使是线性，他也是基于整个participants的linear/sigmoid，而个体是具有独立性的；

  • 遵循特定的函数形式

    1、fixed number of input time points as input

    2、state-based models, 将个体的状态用向量表示

1.2 插值方式探讨

直接删去没有值的内容，会导致信息断层，关于插值信息的处理：

  • 预先插入丢失的数据，后续建模
  • 直接将丢失的数据插入训练，统一整合

4、相关插值方法

  • **”preprocessing strategies(2)”**——插值和训练分开独立，先插值再训练

    • forward filling 采用last timepoint进行插值

  • **“integrative strategy(1)”**——插值和训练集成，一起插值+训练
