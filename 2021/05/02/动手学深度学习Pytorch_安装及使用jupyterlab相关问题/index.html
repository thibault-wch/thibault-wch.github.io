动手学深度学习（Pytorch）-安装及使用jupyter/lab相关问题

一、深度学习介绍

符号学-》 概率模型 -》机器学习 -》深度学习

  • ImageNet 图片分类最大的数据集

  • 物体检测和分割 检测物体位置和像素属于的内容

  • 样式迁移 切换样式

  • 人脸合成

  • 文字生成图片

  • 文字生成模型ssh -L 8831:node02:8091 chenhui@10.190.65.12

  • 无人驾驶

二、永久在集群子节点启动远程jupyter

  • 首先建立一个screen 永久开启jupyter

    screen -S chenhui_jupyter

  • 在对应窗口内，首先在服务器上切换到对应的计算节点

    ssh node02 -X

  • 在服务器上切换对应的conda环境

    conda activate d2l-zh

  • 然后在服务器上输入 jupyter notebook 打开对应端口（默认此时打开8091端口）

    jupyter notebook

  • 在本机cmd采用转发的方式连接，输入服务器密码

​ ssh -L 8831:node02:8812 chenhui@10.190.65.12

  • 在本地采用 http://localhost:8831 即可打开
  • (同理jupyter lab同理；只不过在这里设置为8832)

【备注1】screen方法

1、查看screen的所有窗口

​ screen ls

2、不显示该窗口（后台仍在运行）

​ Ctrl+a+d

3、杀死对应窗口

​ kill 28475

####　４、重新连接上对应的窗口

​ screen -r chenhui_jupyter

【备注2】jupyter 对应端口被占用

如果确定是通过自己一些手段占用，可以采用以下方法

  • 首先通过lsof -i:8888（查询的端口号）
  • 然后通过查询到的端口号对应的pid，采用kill pid的方式杀死对应进程（或者强制杀死kill -9 pid)

【备注3】添加对应kernel在相应的环境中

python -m ipykernel install –name d2l-zh
