LINUX1-基本知识

发表于2021-05-10|更新于2021-05-10|LINUX
|阅读量:

1、预先知识

ll表示显示当前目录文件

2、文件表示

在 Linux 世界里，一切皆文件。

  • reboot 重启
  • useradd 添加用户->添加用户到home目录下
    其中目录结构中：
    usr 上传目录
    mnt 挂载的文件
    opt 放要安装的为念
    /usr/local 放安装后的文件

    3、vi/vim查看

A、正常模式

在正常模式下，我们可以使用快捷键。
以 vim 打开一个档案就直接进入一般模式了(这是默认的模式)。在这个模式中， 你可以使用『上下左右』按键来移动光标，你可以使用『删除字符』或『删除整行』来处理档案内容， 也可以使用『复制、贴上』来处理你的文件数据。

  • ①、所有的 Linux 系统都会内建 vi 文本编辑器。
  • ②、拷贝当前行 yy , 拷贝当前行向下的 5 行 5yy，并粘贴（p）。
  • ③、删除当前行dd, 删除当前行向下的 5 行 5dd
  • ④、在文件中查找某个单词 [命令行下 /关键字 ， 回车 查找 , 输入 n 就是查找下一个 ]
  • ⑤、设置文件的行号，取消文件的行号.[命令行下 : set nu 和 :set nonu]
  • ⑥、编辑 /etc/profile 文件，使用快捷键到底文档的最末行[G]和最首行[gg],注意这些都是在正常模式下执行的。
  • ⑦、在一个文件中输入 “hello” ,然后又撤销这个动作，在正常模式下输入 u

B、插入模式

在模式下，程序员可以输入内容。
按下 i, I, o, O, a, A, r, R 等任何一个字母之后才会进入编辑模式, 一般来说按 i 即可

C、命令行模式

在这个模式当中， 可以提供你相关指令，完成读取、存盘、替换、离开 vim 、显示行号等的动作则是在此模式中达成的！

举例说明:

编辑 /etc/profile 文件，并将光标移动到 第 20 行 shift+g
第一步：显示行号 :set nu
第二步：输入 20
这个数第三步: 输入 shift+g

4、重启与关机

shutdown指令：

  • shutdown -h now ：表示立即关机
  • shutdown -h 1 :表示1分钟后关机
  • shutdown -r now :立即重启
  • shutdown :直接使用，其实就是等价于关机

reboot指令：

  • reboot：重启系统

    相关指令

  • syn：把内存的数据同步到磁盘（关机前进行同步数据）
  • halt：立刻关机

【补充说明】：

登录时尽量少用 root 帐号登录，因为它是系统管理员，最大的权限，避免操作失误。可以利用普通用户登录，登录后再用su - 用户名命令来切换成系统管理员身份.
在提示符下输入 logout 即可注销用户
