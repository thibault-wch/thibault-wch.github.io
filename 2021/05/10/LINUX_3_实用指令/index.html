1、 linux运行级别指令


切换指定运行级别的指令：
基本语法格式

  • （多用户有网络服务）对应 shell状态
  • 如何找回root密码（找回root密码不可以远程操作）
    • 思路：
      进入单用户模式1，修改对应root密码
      （由于进入单用户模式，root不需要密码便可以进行登录）
    • 具体流程：
      开机—>引导时输入e键—>看到一个界面输入e—>看到一个新的界面，选中第二行（编辑kennel）输入e—>切换到这行最后输入1，再输入回车键—>输入b，进入单用户模式，此时使用passwd指令，进行passwd root的修改即可
  • linux运行后如何直接进入命令行界面：
      vim /etc/inittab //将其中的5改成3即可

3、文件目录指令

  • pwd显示当前目录的绝对路径
  • ls指令
    ls [选项] [目录/文件]
  • cd切换指令
    • 绝对路径: 从根目录进行查找定位 /home
    • 相对路径: 从当前目录开始定位 ../home (..代表回到上一级)
  • mkdir 创建目录
    mkdir [选项] 要创建的目录
  • p 创建多级目录
  • rmdir 删除空目录
  • rf 删除有内容的目录
  • touch 创建空文件
    touch 文件名
  • cp 拷贝文件到指定目录
    cp[选项] source des(喜欢相对路径）
    解释：将当前目录的source到拷贝到des
  • r 递归复制整个文件夹
  • rm 删除文件/目录
  • r递归删除整个文件夹
  • f强制删除不提示 可以与其他一起使用
  • mv 移动文件、目录、重命名
  • cat指令
    查看文件内容，以只读的方式
  • n 显示数字编号
    |more ：管道命令；方便分页查看（最好）
  • more指令
    以全屏幕方式按页显示内容：
    相关快捷键：
  • less指令
    显示速度比more快，按页进行加载显示，并不是一次把所有的内容进行加载看大型的文件内容，用less进行相应操作
  • “ > “ 与” >> “指令说明

“ > “ 输出重定向(会覆盖原先文件的内容）
**” >> “** 追加指令(把新的内容追加到尾部）

  • echo 输出内容到控制台
    $PATH指环境变量的路径
  • head 显示文件的开头部分
  • tail 显示文件的尾部部分

【注意】：ctrl+c退出查看的文件内容

  • in指令
    类似于快捷方式（软连接）

in -s [原文件/目录][软链接目录]

cd linkToroot其实最终切换进入的是 /root目录下的内容
pwd查看这个目录会发现它还在这个位置
！！！注意！！！
rm 不能删除指定目录
rm -rf linkToRoot 才能删除软链接内容（不能带/否则资源忙：linkToRoot/)

  • history指令
    可以查看自己之前敲过的历史指令

    4、时间日期类指令

  • data 显示当前时间指令
  • data -s 字符串 设置时间
  • cal 查看当前月的日历

5、搜索查找指令

  • find指令
    find [搜索范围] [选项]
    find指令从指定目录向下递归遍历子目录，将满足条件的文件/目录显示终端
选项 解释
-size 文件大小形式查找 “+”就是“>”，“-”就是“<”, “=”就是不注明；其中单位可根据实际情况进行调整
-name 查找指定文件名
-user 查找对应用户名文件
  • locate指令
    locate 指令可以快速定位文件路径
    locate第一次运行的时候必须先用updatedb
    (因为其基于数据库进行操作）
  • grep指令和管道符号|
    grep过滤查找对应的内容，管道符号”|” 将前一个符号的处理结果传递给后面的命令内容
选项 功能
-n 显示匹配行及行数
-i 忽略字母大小写

6、压缩和解压缩指令

  • gzip/gunzip指令
    gzip压缩文件：不保留源文件 gunzip解压缩文件；
  • zip/unzip指令
    zip 压缩文件

-r：递归压缩，即压缩目录
unzip 进行解压缩文件
**-d<目录> ：指定解压后文件的存放目录**

  • tar指令
    tar指令，打包指令，形成xxx.tar.gz
    tar [选项] xxxx.tar.gz
选项 功能
-c 产生.tar打包文件
-v 显示详细信息
-f 指定压缩后的文件名
-z 打包同时压缩
-x 解包.tar文件
